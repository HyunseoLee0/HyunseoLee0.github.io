My ISP Project

What is the Tetris game?

The Tetris game is a puzzle video game in which falling tetrominoes must be manipulated to form complete lines, which are then cleared from the grid.

Motivation

My ISP group asked ourselves "What is the game that we all played at least once in our lives and enjoyed?", and that answer is TETRIS. This is a popular game and all of our group members have played at one point of our lives. So we decided to create this game ourselves.

Concept

Completing lines by moving differently shaped pieces, which descend onto the playing field. The blocks have to fall in a certain order to gain points which makes the game fun to play. The completed lines disappear and grant the player points.

Code Explanations

There is code to move the block left, down, right using the arrow keys. Also there is code to get points equal to the number of complete lines. The lines will be removed when they are completed.

Conclusions

I had a lot of problems about communicating with group members because I was not good at English, but we can communicate easier using text messages. Also, we have some problems about making the functions about the game, but we can overcome with team members.
